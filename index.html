Routing
›
Load Balancing
Requests route to Cecilia first. If queue > 5, overflow to Gematria. If both busy, queue on Lucidia.
›
Model Selection
API caller specifies model. If model not on target node, request routes to node that has it. Transparent.
›
Failover
Node health checked every 5 min. If inference fails 3x, circuit breaker opens. Traffic routes to next node automatically.
›
No Cloud Fallback
If all 4 nodes are down, requests queue — they DON'T fall back to OpenAI or any cloud provider. Sovereign means sovereign.
